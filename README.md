## 🛠️ How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/subinlearns/c-assignments
   ```

2. Navigate to the project folder:
    ```bash
    cd <Folder-Name> 
    ```

3. Compile the C program:
    ```bash
    gcc 1.c -o 1 -Wall -Wextra -Wpedantic -O2
    ```

4. Run the program:
    ```bash
    ./1
    ```
